# üåç IP Geolocation Tracker

A Python tool designed to track and identify the geographical location of an IP address using external APIs.

## Features
- Retrieves Country, City, and ISP.
- Provides Latitude and Longitude coordinates.
- Simple and fast execution.

## Installation
First, install the required library:
```bash
pip install requests
```
## Usage
Run the script using Python:
```bash
python geo.py
```

[+] Lat/Lon: 21.4913, 39.1841
